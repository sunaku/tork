NAME

tork-herald - reports modified files

SYNOPSIS

tork-herald [OPTION]...

DESCRIPTION

This program monitors the current working directory and all those below it recursively. When any files therein are modified, it prints their relative paths in a single-line JSON array to stdout.

OPTIONS

-h [PATTERN], --help [PATTERN]
Show this help manual and optionally search for PATTERN regular expression.
